MiniLatex Reader

MiniLatexReader is a simple app for displaying LaTeX in a web browser. Just put the LaTeX source text in the div with id = 'source'. You can see how this is done by doing Option-Cmd-U to look at the source text of this web page. One needs (1) a copy of the source text of this web page, 'index.html', (2) 'body.js' and (3) 'Main.min.js'. Of the two javascript files, the first is quite small (1.8K). The second is the compiled Elm app (74K). All the these files, plus the source code for the Elm app can be found on GitHub. Note that the text below scrolls.

Note: when you edit the LaTeX source text, make sure that your editor does not indent it.
